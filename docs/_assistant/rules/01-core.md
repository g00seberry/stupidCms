# Core Principles & Boundaries

-   Docs-first → Code-is-truth: если расхождение, зафиксируй «docs drift» (укажи файл) и сформулируй, что обновить.
-   Laravel 12, PHP 8.2+: `declare(strict_types=1);` повсеместно; PHPStan max.
-   Писать решения в форме «как сделать» (алгоритм + файлы + команды).
-   Все ответы атомарные: можно выполнить без дополнительных «а что если».
-   Отталкивайся от готовых генераторов: ERD, Routes, Permissions.
-   **Обязательно в конце задачи:** запустить `php artisan test` и `composer docs:gen` перед завершением.

**Answer skeleton**

1. Что нужно/контекст (1–2 строки).
2. Где смотреть в репо (пути).
3. Шаги/команды.
4. Критерии готовности/проверки.
